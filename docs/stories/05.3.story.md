## User Story

> Como **gestor de tráfego**, quero **ver todas as fontes de tráfego num dashboard unificado** para que **possa comparar performance entre canais e tomar decisões de investimento em segundos**.

---

## Contexto

Com as integrações Meta e Google Ads activas (05.1 e 05.2), esta story constrói a interface principal do módulo de tráfego. É o ecrã mais utilizado pelos gestores de tráfego da HYPE Flow e pelos account managers em reuniões com clientes.

## Technical Notes

### Queries principais

```typescript
// tRPC procedure: trafego.getDashboard
// Input: { clientId, dateFrom, dateTo, platforms? }
// Output: métricas agregadas + por campanha + heatmap

// Query SQL simplificada:
SELECT
  platform,
  SUM(impressions) as total_impressions,
  SUM(clicks) as total_clicks,
  SUM(leads) as total_leads,
  SUM(spend) as total_spend,
  SUM(spend) / NULLIF(SUM(leads), 0) as cpl,
  date
FROM traffic_metrics
WHERE client_id = $1
  AND date BETWEEN $2 AND $3
GROUP BY platform, date
ORDER BY date ASC
```
