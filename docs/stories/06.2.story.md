## User Story

> Como **gestor comercial**, quero **um calendário centralizado de todas as calls com métricas de performance** para que **possa gerir o pipeline de vendas e optimizar a taxa de conversão por agente**.

---

## Acceptance Criteria

- [ ] **AC1:** View de calendário semanal e mensal em `/calls` com todas as calls agendadas
- [ ] **AC2:** Código de cor por tipo de call: qualificação (azul), proposta (laranja), follow-up (verde)
- [ ] **AC3:** Click num card de call abre painel lateral com detalhe completo + link Meet
- [ ] **AC4:** Vista de lista alternativa (tabela ordenável por data, agente, status, outcome)
- [ ] **AC5:** Métricas no topo: calls esta semana, show-up rate, conversão pós-call, duração média
- [ ] **AC6:** Painel "Briefing Pré-Call": ao abrir uma call futura, mostra histórico da lead, score, últimas interacções
- [ ] **AC7:** Após call, modal de resultado: outcome + notas + próximo passo (gerado automaticamente como task)
- [ ] **AC8:** Filtros: por agente, por cliente, por tipo, por período
- [ ] **AC9:** Indicador de calls em curso (call a decorrer agora → badge verde pulsante)
- [ ] **AC10:** Exportar lista de calls para CSV com todos os campos

## Technical Notes

### Métricas de calls

```typescript
// tRPC: calls.getMetrics
// Show-up rate = calls completed / calls scheduled (excl. cancelled)
// Conversão pós-call = leads que avançaram de fase após call / total calls completed
// Duração média = AVG(actual_duration_min) WHERE actual_duration_min IS NOT NULL
```

### Briefing automático pré-call

```typescript
// Ao abrir painel de call futura:
// 1. Buscar lead_id da call
// 2. Carregar: lead.score, lead.temperature, lead.source
// 3. Carregar: últimas 5 interacções da lead (lead_interactions)
// 4. Carregar: calls anteriores com esta lead
// 5. Renderizar briefing estruturado
```

### Componentes UI

```
/calls
├── CallsHeader           # Métricas + filtros + botão nova call
├── CalendarView          # Calendário semanal/mensal (FullCalendar ou custom)
├── CallListView          # Tabela alternativa
├── CallDetailPanel       # Painel lateral com detalhe
├── PreCallBriefing       # Briefing automático da lead
└── CallOutcomeModal      # Registo do resultado
```
