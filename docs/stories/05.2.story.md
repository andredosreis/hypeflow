## User Story

> Como **gestor de tráfego**, quero **conectar a conta Google Ads de um cliente** para que **as métricas de campanhas Google sincronizem automaticamente** incluindo keywords, grupos de anúncios e quality score.

---

## Contexto

Google Ads é a segunda fonte de tráfego pago mais relevante para os clientes da HYPE Flow, especialmente no nicho de Crédito (alta intenção de pesquisa). Esta story implementa a autenticação OAuth Google e sync de métricas Google Ads.

## Technical Notes

### Google Ads API

**Scopes OAuth necessários:**
```
https://www.googleapis.com/auth/adwords
https://www.googleapis.com/auth/userinfo.email
```

**Google Ads Query Language (GAQL):**
```sql
SELECT
  campaign.id, campaign.name, campaign.status,
  metrics.impressions, metrics.clicks, metrics.cost_micros,
  metrics.conversions, metrics.ctr,
  segments.date
FROM campaign
WHERE segments.date DURING LAST_30_DAYS
ORDER BY metrics.impressions DESC
```

**Keywords query:**
```sql
SELECT
  ad_group_criterion.keyword.text,
  metrics.clicks, metrics.cost_micros,
  metrics.conversions, ad_group_criterion.quality_info.quality_score,
  segments.date
FROM keyword_view
WHERE segments.date DURING LAST_30_DAYS
```

**Nota:** `cost_micros` → dividir por 1.000.000 para obter valor em €
